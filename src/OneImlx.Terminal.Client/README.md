# OneImlx.Terminal.Client

***Open. Flexible. Cross-Platform.***

> **The most flexible cross-platform framework for building modern and secured terminal apps.**

`OneImlx.Terminal.Client` is a cross-platform library for interacting with `OneImlx.Terminal` applications and servers over TCP, UDP, and HTTP.

> ***The full-featured `OneImlx.Terminal` framework is free for educational and evaluation use.***

For commercial or production use, a license is required.
